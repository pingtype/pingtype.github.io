[以色列人紮營的編制] 1 上主這樣指示摩西和亞倫：2 「以色列人紮營的時候，各支派都有指定的區域。每個支派的人都要在自己家族的旗幟下，在聖幕[a]四周稍遠的地方紮營。‎ 3-4 「猶大、以薩迦和西布倫支派的人要在他們家族的旗幟下，在聖幕東面，向着日出的方向紮營。以下是這些支派和他們首領的名字及登記了的軍人數目： 支派 / 首領 / 人數,猶大 / 亞米拿達的兒子拿順 / 74,6005-6 以薩迦 / 蘇押的兒子拿坦業 / 54,4007-8 西布倫 / 希倫的兒子以利押 / 57,4009 這樣，猶大這邊的軍營共有十八萬六千四百人。每當以色列人前往新營地的時候，都由這三個支派開路。‎ 10-11 「呂便、西緬和迦得支派的人要在他們家族的旗幟下，在聖幕南面紮營。以下是這些支派和他們首領的名字及登記了的軍人數目： 支派 / 首領 / 人數,呂便 / 示丟珥的兒子以利蓿 / 46,50012-13 西緬 / 蘇利沙代的兒子示路蔑 / 59,30014-15 迦得 / 丟珥的兒子[b]以利雅薩 / 45,65016 這樣，呂便這邊的軍營共有十五萬一千四百五十人。每當以色列人前行，這三個支派都走在第二隊。‎ 17「隨後，利未人要抬着聖幕從營地中間出發，其他支派也要按照紮營的次序出發，在自己家族的旗幟下前進。‎ 18-19「以法蓮、瑪拿西和便雅憫支派的人要在他們家族的旗幟下，在聖幕西面紮營。以下是這些支派和他們首領的名字及登記了的軍人數目： 支派 / 首領 / 人數,以法蓮 / 亞米忽的兒子以利沙瑪 / 40,50020-21 瑪拿西 / 比大蓿的兒子迦瑪列 / 32,20022-23 便雅憫 / 基多尼的兒子亞比但 / 35,40024 這樣，以法蓮這邊的軍營共有十萬八千一百人。每當以色列人前行，這三個支派都走在第三隊。‎ 25-26 「但、亞設和拿弗他利支派的人要在他們家族的旗幟下，在聖幕北面紮營。以下是這些支派和他們首領的名字及登記了的軍人數目： 支派 / 首領 / 人數,但 / 亞米沙代的兒子亞希以謝 / 62,70027-28 亞設 / 俄蘭的兒子帕結 / 41,50029-30 拿弗他利 / 以南的兒子亞希拉 / 53,40031 這樣，在但這邊的軍營共有十五萬七千六百人。每當以色列人前行，這三個支派都走在最後面。」‎ 32 總的來說，按照家族列出來的以色列軍隊總數為六十萬三千五百五十人。33 按照上主的命令，利未人沒有計算在內。34 以色列民遵照上主對摩西的命令去做，各宗族和家族都在自己的旗幟下紮營、行進，正如上主指示他們的那樣。‎ Footnotes:a.   2:2 希伯來文為「會幕」，也見於民二17。 b.   2:14-15 與許多希伯來文抄本、《撒馬利亞五經》、《武加大拉丁譯本》相同（另見民一14）；大部分希伯來文抄本為「流珥的兒子」。
