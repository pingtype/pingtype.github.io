HubText is a text editor for Github.

Type the filename and click Open. Make your edits, and click Save.

You are probably trying to edit zhuyin.txt (bopomofo) or commonWords.txt (common words). You can also edit your own files saved using Save HTML or Save Text. Please don't edit the dictionary directly; use the Dictionary Editor for that.
