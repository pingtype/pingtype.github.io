Pingtype English is a literal translator for English-to-Chinese. It adds colours for nouns/verbs/adjectives, shows IPA/Phonic/Bopomofo pronunciation, and translates each word literally to Chinese.

It is based on Pingtype Chinese, which goes the other way, from Chinese to English.

Text output

If you want to copy-paste your text to another place (e.g. music lyrics), you may prefer to have it in plain text. The text area on the top-right has the same output in plain text.

Memory

Suppose you want to save a sentence for later. The MC, M+, and MR buttons work like a calculator.

M+ appends the current phrase to the memory. It does not erase what is already saved.

MR loads all the memory into the Chinese input text area.

MC clears the memory, so you lose everything you saved.

Clear only clears the view, it does not clear the memory. You can use MR to recall the saved phrases.

Editing the dictionary

I need you!

My dictionary is not very good yet. I need to add words. I need to edit words. Sometimes I need to remove words. This is all possible using the Dictionary Editor. Please use it, and don't forget to upload! First press the Edit button to open the floating window. You can then drag that around your screen using the mouse.

If a word does not exist, then please remove it. Copy-paste the word into the Remove text area, and click the Remove button. Then click Retranslate. Sometimes you have to do this more than once.

You can also add a word to the dictionary. Copy-paste the word into the Add textarea. Don't worry if there are extra tabs; you can remove them by clicking the Search üîç button. This also looks up the word in the dictionary, in case you want to edit an existing word.

You can click the "English To IPA" button to get the pronunciation from Lingorado. Next, click the IPA to Phonics and IPA to Bopomofo buttons.
Then type the Chinese word that has the same meaning as the English word, and click Add. Then click Retranslate, and see that the word has been combined into one.

Finally, choose the POS from the list: Other/Noun/Verb/Adjective. The Classify button will try to search a dictionary to find the POS automatically.

You can also edit definitions using the Add button. If the word already exists, the new definition will be used.
