Unitype was some research I did to try to make every Unicode character into a four-letter word.

Instead of typing hex codes, I mapped the 16 hex values to keys on the qwerty keyboard. You have to press four keys for every character, but that's still often faster than trying to type Chinese any other way. Existing methods require reading and choosing a character from a list on the screen, which sucks if you're blind.

Pingxing doesn't support it yet, so I'll only write about it when I port the code to the main translator app.
