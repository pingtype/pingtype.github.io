Maps

Google removed Pinyin from Google Maps in Taiwan on April 26th.

I made an alternative, because people are asking for it.

It's really not ready for public release, but sending bug reports to Google has fallen on deaf ears. If you're travelling in Taiwan, please feel free to use it.

If you're a programmer and want to help improve the GUI, please contact me.

Offline for mobile

Download Galileo, add Maps from PC. Download map tiles using MOBAC and a custom tile source, put sqlitedb file into app using iTunes or a file browser e.g. DiskAid.

Offline for desktop

Download BAMB, set up a local web server. Download map tiles in Galileo format using MOBAC and a custom tile source. Put sqlitedb files into bamb/Maps folder.
